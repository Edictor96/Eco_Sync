EcoSync Mobile (Expo) - Quick Start
----------------------------------
1. Install Node.js and npm.
2. Install Expo CLI globally: npm install -g expo-cli
3. In this folder, run: npm install
4. Edit src/api.js and set BASE = 'https://<YOUR_RENDER_BACKEND_URL>' after deploying backend to Render.
5. Start dev server: expo start
6. Use Expo Go app on Android to scan QR and run the app, or use an emulator.

To build a standalone APK/AAB:
- Follow Expo EAS Build docs: https://docs.expo.dev/build/overview/
